# caddy

[![Build](https://github.com/gsmlg-ci/caddy/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/caddy/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/caddy`
- `ghcr.io/gsmlg-dev/caddy`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/caddy:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/caddy:latest
```

## Build

```bash
docker build -t gsmlg/caddy .
```
